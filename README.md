This is a simple Random Color Generator web application built using basic HTML, CSS, and JavaScript. The project demonstrates fundamental concepts of DOM manipulation and event handling in JavaScript.

The application allows users to generate a random color by clicking a button. When clicked, the background color of the display box changes dynamically, and the generated RGB color value is shown on the screen.

The structure of the project is kept minimal using HTML for layout , CSS for basic styling , and JavaScript for adding interactivity.
